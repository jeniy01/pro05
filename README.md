# 숫자 맞추기 게임
http://127.0.0.1:5500/guess.html

-------------------------------------------------------------------------------------------------
#### 사용자가 링크에 접속한 후, 숫자란에 원하는 숫자를 입력하여 컴퓨터가 생성한 숫자를 맞추는 게임
1. 힌트는 up & down <br>
2. 결과란에 입력받은 숫자보다 크거나 작은 값을 요구할 수 있음 <br>
3. 같은 숫자로 재검사시, 추측 횟수가 1씩 증가됨 <br>
4. 다시시작을 누르면 값이 초기화 되면서 재시작 가능 <br>
